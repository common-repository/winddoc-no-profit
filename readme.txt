== Description ==
Alcune associazioni hanno l’obbligo o la volontà di pubblicare la lista dei soci iscritti alla loro associazione.

Grazie a questa estenzione puoi collegare il tuo account di WindDoc No-Profit e visualizzare la lista dei soci attivi della tua associazione direttamente sul tuo sito web.
